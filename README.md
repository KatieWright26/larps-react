
# larps

it's a real sweet app

## how

### install

```
git clone https://github.com/katiewright26/larps-react
cd
npm install
```

## develop

```
npm start
```
browse to <http://localhost:9966/>.

to run the backend server, follow instructions at https://github.com/katiewright26/react-larps-api

## test

```
npm test
```

## deploy

```
npm run deploy
```
